# Maintainers

## Primary Maintainer

- GitHub: [@overdosesol](https://github.com/overdosesol)
- Role: project owner and primary maintainer

## Maintainer Responsibilities

- Review issues and pull requests.
- Keep dependencies and security alerts under control.
- Maintain release notes and deployment documentation.
- Review AI-assisted changes before they land.
- Keep production secrets and operator notes out of the public repository.

## AI-Assisted Maintenance

Codex and OpenAI API models may be used for codebase understanding, review
assistance, security hardening, documentation work, release preparation and
maintenance automation.

AI output is not accepted automatically. The maintainer remains responsible for
the final diff, tests, security decisions and releases.
